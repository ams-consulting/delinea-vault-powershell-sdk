# Delinea Vault PowerShell SDK
This SDK is an unofficial PowerShell module for Delinea Vault (aka Secret Server). The module provides wrapper functions for the Delinea Vault API as PowerShell Cmdlets that can be used from PowerShell scripts or from an interactive PS Session.

# Installation
Run the Install-DelineaVaultSDK.ps1 script with Local Admin privilege. Script will install this PowerShell module under C:\Program Files\Delinea\PowerShell by default.

# Usage
For more information on Cmdlets usage and sample scripts, use Get-Help command on each Cmdlet (Wiki page is WIP)
